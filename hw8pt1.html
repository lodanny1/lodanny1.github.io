Homework 8 - AJAX Application



Explanation of AJAX

This page uses AJAX to fetch the xml file without reloading the page. The XMLHttpRequest object is used to send a GET request to the server. When the file is loaded, it triggers a function that parses the XML data.

The responseXML property allows the JavaScript code to access the XML content, then it displays all information such as: ARTIST, TITLE, COUNTRY, COMPANY, PRICE, and YEAR, and inserts it into an HTML table.

REST API

For this project, I chose the QRCode Monkey API, a rest api that generates customizable QR codes. The API accepts input in JSON format, which specifies the data, QR code design, and output format.

To use the API, an API key is required. It offers a free tier for basic usage, while advanced features like higher request limits and custom logos require a premium subscription.

You can access the API documentation here: QRCode Monkey API Documentation.
